# Examples

This folder contains fully completed example implementations of the AI Business Template Kit — fictional businesses with every template file filled in.

Use these to:
- Understand what a finished set of context files looks like
- See how the templates work together as a system
- Copy the structure and adapt it to your own business
- Train new team members on what good context looks like

---

## Available Examples

### [Folio — Client Portal SaaS for Creative Agencies](./folio/)

A B2B SaaS company that helps creative agencies (branding studios, web design, video production) manage client projects, share deliverables, and collect approvals — replacing the chaos of email, Dropbox, and Notion threads.

**Good reference for:** SaaS businesses, B2B products, agencies selling to agencies, subscription pricing models.

---

## How to Use an Example

1. Browse the example's `context/` files to see how each template is filled in
2. Read the `prompts/MASTER_PROMPT.md` to see what a finished master prompt looks like
3. Use it as a reference while filling in your own templates — open the example alongside your blank template

**Don't copy it directly.** The example is fictional. The value is in seeing the structure and level of detail — not in reusing the content.
